## Libraries Used

**React Router DOM**  
- Used for managing client-side routing and navigation between pages (`Search`, `MySchedule`, and `Register`).  
- It provides a simple way to handle dynamic rendering of views without reloading the page, keeping the experience smooth and consistent.  

**Tailwind CSS**  
- Chosen for its utility-first approach, allowing rapid styling directly within JSX while maintaining consistency.  
- It helps keep the codebase lightweight and avoids the need for external CSS files or complex naming conventions.
